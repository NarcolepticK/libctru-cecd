# libctru-cecd
CECD service code for libctru

Requires existing devkitPro install

This aims to fill in the missing service code, and enable CECTool to function.

## CECD:u Functions
- Open
- Read
- ReadMessage
- ReadMessageWithHMAC
- Write
- WriteMessage
- WriteMessageWithHMAC
- Delete
- SetData
- ReadData
- Start
- Stop
- GetCecdState
- GetCecInfoEventHandle
- GetChangeStateEventHandle
- OpenAndWrite
- OpenAndRead

## Partially Supported
- GetCecInfoBuffer
- GetEventLog
- GetEventLogStart
